# SLEEP

**BASIC Set:** Extension

## Syntax
```
SLEEP seconds
```

## Description
Pauses program execution for the specified number of seconds.

## Parameters
- `seconds`: Number of seconds to pause (can be fractional)

## Example
```
10 PRINT "Wait..."
20 SLEEP 2
30 PRINT "Done!"
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Useful for timing and delays.
