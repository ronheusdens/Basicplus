# GET

**BASIC Set:** Extension

## Syntax
```
GET #n, variable
```

## Description
Reads a record from a file into a variable.

## Parameters
- `#n`: File number
- `variable`: Variable to store the data

## Example
```
GET #1, A$
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Used for random-access files.
