# PAINT

**BASIC Set:** Extension

## Syntax
```
PAINT (x, y)
```

## Description
Fills an area starting at the specified coordinates.

## Parameters
- `x`: X coordinate
- `y`: Y coordinate

## Example
```
PAINT (10, 20)
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Used for graphics operations.
