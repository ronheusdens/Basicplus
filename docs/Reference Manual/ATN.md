# ATN() Function

## Overview
Returns the arctangent (inverse tangent) of a number, in radians.

## Syntax
```basic
result = ATN(x)
```

## Parameters
- **x**: Value whose arctangent is to be calculated

## Return Value
- Angle in radians

## Examples
```basic
10 PRINT ATN(1)        ' Output: ~0.785 (PI/4)
20 PRINT ATN(0)        ' Output: 0
```

## Notes
- Result is in radians
- To convert radians to degrees: degrees = radians * 180 / 3.14159
