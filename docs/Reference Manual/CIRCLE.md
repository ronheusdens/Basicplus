# CIRCLE

**BASIC Set:** Extension

## Syntax
```
CIRCLE (x, y), r
```

## Description
Draws a circle at the specified coordinates with radius r.

## Parameters
- `x`: X coordinate
- `y`: Y coordinate
- `r`: Radius

## Example
```
CIRCLE (40, 12), 5
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Used for graphics operations.
