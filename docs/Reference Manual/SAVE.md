# SAVE

**BASIC Set:** Extension

## Syntax
```
SAVE "file.bas"
```

## Description
Saves the current BASIC program to the specified file.

## Parameters
- `"file.bas"`: The filename to save to (must be in quotes)

## Example
```
SAVE "MYPROG.BAS"
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Overwrites the file if it already exists.
