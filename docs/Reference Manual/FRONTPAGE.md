## About This Manual

This manual provides comprehensive documentation for the TRS-80 Level II BASIC interpreter. Each command is documented with syntax, description, parameters, examples, and notes. Commands are marked as either **Level II BASIC** (standard) or **Extension** (interpreter-specific enhancements).

**Audience:** Users of all skill levels, from beginners to experienced programmers.
