# LINE

**BASIC Set:** Extension

## Syntax
```
LINE INPUT #n, variable
```

## Description
Reads a line of text from a file into a variable.

## Parameters
- `#n`: File number
- `variable`: Variable to store the line

## Example
```
LINE INPUT #1, A$
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Reads up to a newline character.
