# REM

**BASIC Set:** Level II BASIC

## Syntax
```
REM comment
```

## Description
Marks the rest of the line as a comment. The interpreter ignores everything after REM.

## Parameters
- `comment`: Any text (ignored by interpreter)

## Example
```
10 REM This is a comment
15 'It is also possible just to use an apostrophe to place a remarkt on a single line
20 PRINT "Hello" 'And also on a statement line
```

## Notes
- REM can be abbreviated as a single quote `'` in some dialects, as in our Extended Level II BASIC.
