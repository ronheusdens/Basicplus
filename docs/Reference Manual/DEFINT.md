# DEFINT

**BASIC Set:** Level II BASIC

## Syntax
```
DEFINT letter-range [, ...]
```

## Description
Declares variables with names starting with specified letters as integers by default.

## Parameters
- `letter-range`: Range of letters (e.g., A-Z)

## Example
```
DEFINT A-C, X, Z
```

## Notes
- Applies to all variables starting with those letters.
