# PUT

**BASIC Set:** Extension

## Syntax
```
PUT #n, variable
```

## Description
Writes a record from a variable to a file.

## Parameters
- `#n`: File number
- `variable`: Variable containing the data

## Example
```
PUT #1, A$
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Used for random-access files.
