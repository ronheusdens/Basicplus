# RESET

**BASIC Set:** Extension

## Syntax
```
RESET (x, y)
```

## Description
Clears a pixel at the specified coordinates.

## Parameters
- `x`: X coordinate
- `y`: Y coordinate

## Example
```
RESET (10, 20)
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Used for graphics operations.
