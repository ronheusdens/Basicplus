# COLOR

**BASIC Set:** Extension

## Syntax
```
COLOR n
```

## Description
Sets the foreground color for text or graphics.

## Parameters
- `n`: Color code (valid range: 0â€“7; 0=black, 1=blue, 2=green, 3=cyan, 4=red, 5=magenta, 6=yellow, 7=white)

## Example
```

```

## Notes
- Currently only works in graphics-mode (SCREEN = 1)
- Not part of Level II BASIC; provided as an extension.
- Color codes depend on system implementation.
