# DEFSTR

**BASIC Set:** Level II BASIC

## Syntax
```
DEFSTR letter-range [, ...]
```

## Description
Declares variables with names starting with specified letters as strings by default.

## Parameters
- `letter-range`: Range of letters (e.g., A-Z)

## Example
```
DEFSTR A-C, X, Z
```

## Notes
- Applies to all variables starting with those letters.
