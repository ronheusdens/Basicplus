# WHILE

**BASIC Set:** Extension

## Syntax
```
WHILE condition
  ...
WEND
```

## Description
Begins a loop that continues while the condition is true.

## Parameters
- `condition`: Expression to evaluate

## Example
```
WHILE A < 10
  PRINT A
  A = A + 1
WEND
```

## Notes
- Not part of Level II BASIC; provided as an extension.
- Must be paired with WEND.
